# Next.js template

This is a Next.js template with shadcn/ui.

## Adding componentss

To add componentss to your app, run the following command:

```bash
npx shadcn@latest add button
```

This will place the ui componentss in the `componentss` directory.

## Using componentss

To use the componentss in your app, import them as follows:

```tsx
import { Button } from "@/componentss/ui/button";
```
